Project Brief

Emily was approached by Start Mesh to create a brand identity, including a logo, business card design and website.

Logo Design & Business Cards

Start Mesh wanted a unique, memorable logo that could be easily adapted into an app icon. The simple, strong icon portrays the mesh concept while also representing networking throughout the globe. Business card designs were created for each of the team members.

Website Design

Using the newly created brand and logo, Emily designed the Start Mesh website. The website design is clear and without clutter, focusing on Start Mesh’s key information. Emily works closely with the Start Mesh team adding additional sections to improve the website.
